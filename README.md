## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running clone this repository using:

```sh
  git clone https://github.com/paulshaun-kdtk/Ruby_Capstone_Catalogue_Of_Things.git
  cd Ruby_Capstone_Catalogue_Of_Things
```

### Prerequisites

In order to run this project you need to install:

> Ruby
> rspec

### Setup
Clone this repository to your desired folder:

```sh

    cd Ruby_Capstone_Catalogue_Of_Things
```
